A simple example of a {{ page.title }} descriptor looks as follows.

```
{
  "name": "my-unique-datapackage",
  "title": "my Unique Data Package",
  "licenses": [
    "http://example.com/license.md"
  ],
  "resources": [
    {
      "name": "",
      "path": ""
    }
  ]
  "sources": [
    {
      "name": ""
    }
  ]
}
```
